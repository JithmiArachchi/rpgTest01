# Customer Management System (CMS)

A modern full-stack **Customer Management System** built with **Spring Boot** that seamlessly integrates with **IBM i (AS/400)** database.

## 🚀 Project Overview

This project demonstrates how to build a modern web application that connects to a traditional **IBM i** system using **JT400** driver. It allows users to perform full CRUD operations (Create, Read, Update, Delete) on customer data stored in IBM i DB2 database.

### Key Features
- View all customers
- Add new customer
- Update customer details
- Delete customer
- RESTful APIs with Spring Boot
- Clean and responsive Bootstrap UI
- Direct connection to IBM i (PUB400)

### Technology Stack
- **Backend**: Java 17 + Spring Boot 3
- **Database**: IBM i DB2 (via JT400)
- **Frontend**: HTML + Bootstrap 5 + JavaScript
- **Build Tool**: Maven

### How to Run

1. Clone the repository
2. Update `application.properties` with your IBM i credentials
3. Run the Spring Boot application
4. Open browser → `http://localhost:8089`

### Learning Outcomes
- IBM i database connectivity using JT400
- Spring Boot REST API development
- CRUD operations with legacy IBM i systems
- Modern frontend integration with backend
